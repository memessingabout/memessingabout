History & Versions of CSS

CSS (Cascading Style Sheets) styles HTML content, separating presentation from structure. Developed by Håkon Wium Lie and Bert Bos, it has evolved through several versions:

  • CSS1 (1996): Basic styling (e.g., fonts, colors, margins).
  • CSS2 (1998): Added positioning, pseudo-classes, and media types.
  • CSS2.1 (2011): Clarified and standardized CSS2.
  • CSS3 (1999–present): Modular, introduced Flexbox, Grid, animations, transforms, and responsive design features.
  • CSS4 (conceptual): No official version; refers to new modules (e.g., advanced selectors).
  • Current Standard: Managed by W3C, with ongoing module development (e.g., CSS Color, CSS Grid).

Notes: Use CSS3 modules for modern development. Avoid deprecated properties (e.g., font-size-adjust in older browsers).

Basic CSS Syntax

CSS consists of rules with selectors and declarations.

Concept Description Syntax/Example
Rule Selector and declaration block. 
selector {
    property: value;
}
Selector Targets HTML elements. 
p { color: blue; }
Declaration Property-value pair. 
color: blue;
Comment Non-rendering notes. 
/* This is a comment */
Inclusion Methods to include CSS. 
/* External CSS */
<link rel="stylesheet" href="styles.css">

/* Internal CSS */
<style>
    p { color: blue; }
</style>

/* Inline CSS */
<div style="color: blue;">

Notes: Prefer external CSS for maintainability. Avoid inline CSS for large projects.

CSS Selectors

Selectors target HTML elements for styling.

Selector Type Description Example
Element Targets HTML tags. 
p { color: blue; }
Class Targets elements with a class. 
.highlight { color: red; }
ID Targets a unique element. 
#main { background: #f0f0f0; }
Attribute Targets elements with specific attributes. 
[type="text"] { border: 1px solid; }
Universal Targets all elements. 
* { margin: 0; }
Pseudo-class Targets elements in a state (e.g., hover). 
a:hover { color: red; }
Pseudo-element Targets parts of elements. 
p::first-line { font-weight: bold; }
Combinators Relationships between elements. 
/* Descendant */
div p { color: blue; }

/* Child */
div > p { color: green; }

/* Adjacent Sibling */
h2 + p { margin-top: 0; }

/* General Sibling */
h2 ~ p { color: gray; }

Notes: Use specific selectors to avoid conflicts. Pseudo-classes like :hover, :focus, and :nth-child enhance interactivity.

CSS Properties

Key properties for styling elements.

Category Properties Example
Text color, font-size, font-family, font-weight, text-align, text-decoration, line-height, text-transform 
p {
    color: #333;
    font-size: 16px;
    font-family: Arial;
    text-align: center;
}
Box Model margin, padding, border, width, height, box-sizing 
div {
    margin: 10px;
    padding: 20px;
    border: 1px solid #ccc;
    box-sizing: border-box;
}
Background background-color, background-image, background-size, background-position 
body {
    background-color: #f4f4f4;
    background-image: url('bg.jpg');
}
Display display (block, inline, inline-block, none, flex, grid) 
div { display: flex; }
Positioning position (static, relative, absolute, fixed, sticky), top, left, z-index 
.sticky {
    position: sticky;
    top: 0;
    z-index: 100;
}
Typography font-style, letter-spacing, word-spacing, text-shadow 
h1 {
    font-style: italic;
    text-shadow: 2px 2px 4px #000;
}
Colors color, background-color, opacity, rgb, rgba, hsl, hsla 
button {
    background-color: rgba(76, 175, 80, 0.8);
}
Transitions transition, transition-property, transition-duration 
button {
    transition: background-color 0.3s ease;
}
Animations animation, @keyframes 
@keyframes fade {
    from { opacity: 0; }
    to { opacity: 1; }
}
div { animation: fade 2s; }
Transforms transform (rotate, scale, translate) 
div {
    transform: rotate(45deg);
}

Notes: Use box-sizing: border-box for predictable sizing. Combine properties for efficiency (e.g., margin shorthand).

Layout Techniques

Modern CSS layouts for responsive design.

Technique Description Example
Flexbox One-dimensional layout (row or column). 
.container {
    display: flex;
    gap: 20px;
    justify-content: space-between;
    align-items: center;
}
Grid Two-dimensional layout (rows and columns). 
.grid {
    display: grid;
    grid-template-columns: 1fr 1fr;
    gap: 10px;
}
Float Legacy layout for floating elements. 
.float {
    float: left;
    width: 50%;
}
Positioning Controls element placement. 
.fixed {
    position: fixed;
    top: 0;
    width: 100%;
}
Media Queries Responsive design for different screens. 
@media (max-width: 600px) {
    body { font-size: 14px; }
}

Notes: Prefer Flexbox or Grid over float. Use media queries for responsive design.

Advanced CSS Features

Modern features for dynamic and complex styling.

Feature Description Example
Variables Custom properties for reusable values. 
:root {
    --primary-color: #4CAF50;
}
button { background: var(--primary-color); }
Pseudo-elements Style parts of elements (e.g., ::before). 
p::before {
    content: "★";
    color: red;
}
CSS Functions Dynamic calculations (e.g., calc, min, max). 
width: calc(100% - 20px);
Filters Apply visual effects (e.g., blur, grayscale). 
img { filter: grayscale(50%); }
Gradients Linear or radial color transitions. 
background: linear-gradient(to right, red, blue);
Responsive Units Units like vw, vh, rem, em. 
font-size: 2rem;

Notes: Use variables for maintainability. Test advanced features for browser compatibility.
